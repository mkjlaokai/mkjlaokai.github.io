1.4、COLO虚拟机启动流程

1.4.1、COLO虚拟机启动

%% 时序图例子,-> 直线，-->虚线，->>实线箭头
  sequenceDiagram
    participant PVM
    participant SVM
    PVM --> PVM: 开启PVM
    SVM --> SVM: 开启SVM
    SVM ->> PVM: SVM 开启NBD服务
    PVM ->> SVM: PVM开启COLO迁移
    note over PVM,SVM: 正常运行

1.4.2、辅虚拟机故障恢复

%% 时序图例子,-> 直线，-->虚线，->>实线箭头
  sequenceDiagram
    participant PVM
    participant SVM
    note over PVM,SVM: 模拟SVM故障
    PVM --> PVM: 移除object,手动停止COLO服务
    PVM --> SVM: PVM正常运行
    SVM --> SVM: 再次启动SVM
    loop 同步磁盘
    	PVM --> PVM: 同步磁盘
    end
    PVM --> PVM: 添加object
    PVM ->> SVM: PVM开启COLO迁移

1.4.3、 主虚拟机故障恢复

%% 时序图例子,-> 直线，-->虚线，->>实线箭头
  sequenceDiagram
    participant PVM
    participant SVM
    note over PVM,SVM: 正常运行
    note over PVM,SVM: 模拟PVM故障
    SVM --> SVM: 删除网卡重定向的设备,停止COLO服务
    SVM --> SVM: 准备作为主虚拟机需要的资源
    SVM --> SVM: SVM正常运行
    PVM --> PVM: 以SVM身份启动PVM
    loop 同步磁盘
    	SVM --> SVM: 同步磁盘
    end
    SVM --> SVM: 添加object
    SVM ->> PVM: SVM开启COLO迁移
    note over PVM,SVM: 正常运行，身份转化
    note over PVM,PVM: SVM
    note over SVM,SVM: PVM
